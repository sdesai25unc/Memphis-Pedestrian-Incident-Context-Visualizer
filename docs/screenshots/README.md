# Screenshots

These images are referenced by the main `README.md` "Screenshots" section with
relative paths, so they render on GitHub the moment the files exist here.

## How to refresh them (2-minute task)

Take the screenshot, name it **exactly** as below, and drop it in this folder
(overwrite the old one). No README edits are needed — the paths already point here.

| Filename | What to capture |
|---|---|
| `01-map-overview.png` | The full interactive map (crashes colored by road owner, with the search panel visible). |
| `02-search-count-a.png` | A search or map-click result card — showing the snapped road, **Owner**, and the crashes on the ±300 m stretch (with the orange ±window bars visible). |
| `03-city-vs-state.png` | The **"See City vs State segments"** view — a corridor (e.g. Poplar) colored teal = City of Memphis, crimson = TDOT/State. |
| `04-dashboard.png` | The findings dashboard + the 25 deadliest-corridors table. |

Recommended width ~1400px (PNG). Keep the same filenames each time so the README
never has to change.
